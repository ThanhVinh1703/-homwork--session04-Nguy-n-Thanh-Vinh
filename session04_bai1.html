What is CSS

Casscading Style Sheet (CSS) is used to format the layout of the webpage

With CSS, you can control the color ,font and the size of text, the spacing between elements, how element are positioned and laid out, what background images or background colors are to be used, dfferent displays for different devices and creen sizes, and much more!


Tip: The word Casscading means that a style applied to parent element will also aplly to all children element with the in parent. So, if you set the color of the body text to "blue", all headings, paragraphs, and other text element within the body will also get the same color( unless you specify somethingalse)!
